# Harvester

The correct software architecture for OAI harvesters

#### Objectives

###### tech:

- [ ] Golang

- [ ] PostgreSQL (Best Solid Open Source Database)

- [ ] ElasticSearch (Best search engine and storage)

- [ ] Cloud ready, scalable

- [ ] Non-blocking Async jobs.

###### workers:

- [ ] OAI Harvester

- [ ] Dublin Core Parser

- [ ] Elastic Indexer

###### endpoint:

- [ ] REST Search api

- [ ] Workers Triggers

- [ ] Cron job api

###### management:

- [ ] Admin UI
